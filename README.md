# react-project
    react 模仿大众点评app react + react-router4

## Build Setup

``` bash
# 下载或克隆下来，然后安装依赖
npm install

#启动本地接口服务
npm run mock

# 开发预览
npm run start

# 打包发布，生成的文件在dist文件夹中
npm run build
```
